# Event ID 4688 - Process Creation Investigations

This folder contains investigations related to Windows Security Event ID 4688 (Process Creation).

## Objectives

- Analyze newly created processes.
- Identify parent-child process relationships.
- Review process command-line arguments.
- Determine whether activity is expected or suspicious.
- Document investigation findings using SOC analyst methodology.
